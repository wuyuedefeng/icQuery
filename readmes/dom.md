* `$ic()`

### 管理类
* `addClass`  // 添加类
* `removeClass` // 移除类
* `replaceClass` // 替换类 （先移除后添加）
* `toggleClass` // 存在移除，不存在添加类
* `containsClass` // 判断是否包含类

* `offsetTop`
